| T-03-02 | Information Disclosure | Notes/descope-reason textarea content visible in DOM | accept | No authentication in this application (per CLAUDE.md: "No authentication is implemented"). All users on the network are assumed to have equivalent access. No PII concern. |
| T-03-03 | Spoofing | Modal renders above all other content via z-50 | accept | z-index is presentational. No spoofing vector — the modal does not impersonate external sites. |
| T-03-04 | Tampering | notes and descopeReason sent to backend without sanitization | mitigate | Backend (FastAPI + Pydantic) already validates and persists to Markdown files.
